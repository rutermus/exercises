## Упражнение

`solution.js`

Напишите программу, которая выводит на экран:

```
- Did Joffrey agree?
- He did. He also said "I love using \n".
```

При этом программа использует только один `console.log()`, но результат на экране должен выглядеть в точности, как показано выше.
